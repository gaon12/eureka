# Node.js 서버 API
아파트 관리 프로그램 유레카 프로젝트의 대부분의 기능을 담당하고 있는 HTTP 기반의 REST API입니다.

## 서비스 접속
### 요청
|HTTP|
|--|
|GET http://{address}:{port}/|

### 응답
#### 응답 바디
|데이터 타입|설명|
|--|--|
|html|정적 파일|

## 회원 가입
### 요청
|HTTP|
|--|
|POST http://{address}:{port}/user/signup/|
#### 요청 헤더
|헤더명|설명|
|--|--|
|Content-Type|바이너리 전송 형식<br>```Content-Type: application/json```|
#### 요청 바디
|필드성 성공"
    }
```

### 오류
#### 오류 예시
```JSON
    {
        "status": 400,
        "error": {
            "errorCode": "E400",
            "message": "필수 항목 미입력"
        }
    }
```

# 오류 코드
|HttpStatusCode|ErrorCode|ErrorMessage|
|--|--|--|
|400|E400|필수 항목 미입력|
|400|E401|아이디 or 비밀번호 오류|
|400|E402|비밀번호 불일치|
|400|E403|등록되지 않은 사용자|
|400|E404|세션 정보 없음|
|400|E405|이미 존재하는 회원|
|400|E410|등록되지 않은 차량|
|400|E411|이미 등록 된 차량|
|403|E420|관리자 권한 필요|
|500|E500|서버 에러|
|500|E501|차량 등록 실패|
